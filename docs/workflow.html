Test outputs

  • Figures generated by tests go to tests/figures/<test-file>/.
  • Generated NetCDF and other non-figure outputs go to tests/generated/.
  • Reference fixtures remain under tests/data/ and are not overwritten.

Scientific code boundaries

Workflow, packaging, documentation, typing, tests and CI can evolve. Scientific algorithm changes should remain isolated and validated through equivalence tests.
